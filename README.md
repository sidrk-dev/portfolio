# Portfolio - Siddharth Rajasekaran

This is a personal engineering portfolio website built with **Next.js 15**, **Tailwind CSS**, and **Framer Motion**.

## Design System
- **Dark Theme Only**: Background `#0a0a0a`.
- **Primary Color**: Electric Blue (`#3B82F6`) to Cyan (`#06B6D4`) gradients.
- **Typography**: Inter (Body), Space Grotesk (Headings).
- **Style**: Glassmorphism, animated circuit board particles, glowing effects.

## Features
- **Hero**: Split-text name reveal and text-scramble role cycle.
- **About**: Animated statistics.
- **Projects**: Data-driven project cards with hover effects.
- **Skills**: Categorized skill badges.
- **Experience**: Vertical timeline with glowing nodes.
- **Contact**: Simple social links.

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run the development server**:
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

3. **Build for production**:
   ```bash
   npm run build
   ```

## Customization
- Update content in `src/data/portfolio.ts`.
- Adjust styles in `src/app/globals.css`.

## Deployment
This project is optimized for deployment on **Vercel**.
1. Push to GitHub.
2. Import project in Vercel.
3. Deploy.
